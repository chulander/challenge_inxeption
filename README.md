# Installation

### Clone Repository

```bash
git clone git@github.com:chulander/inxeption.git
```

### Install Dependencies

```bash
cd inxeption
npm i
```

### Setup Environemental Variables

```bash
touch .env
```

DB_NAME=ANY_STRING

### Run Test

```bash
npm test
```

### Dev Server

```
npm run dev
```

### Production

```bash
npm run build
npm run prod
```
